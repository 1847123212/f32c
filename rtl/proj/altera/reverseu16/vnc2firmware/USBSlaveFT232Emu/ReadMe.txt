USBSlave FT232 Emulation Readme
-------------------------------

This project demonstrates configuring the VNC2 to function similar to a FT232 USB Slave IC from FTDI USB to UART in a dual-threaded application.  The following interfaces are used:  USBSlave, FT232Slave, GPIO, UART, DMA

The VNC2 is connected to the host (PC) via USB Port 1.

Restrictions:
Serial BREAK not supported
